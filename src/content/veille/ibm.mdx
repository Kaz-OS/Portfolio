# Résumé 
Résumé : Ce document traite de l'acquisition de DataStax par IBM afin d'améliorer les capacités de watsonx.ai. 
DataStax, qui propose une distribution commerciale de Apache Cassandra, est réputée pour sa gestion des bases de données NoSQL et de la recherche vectorielle. 
L'objectif d'IBM est d'intégrer ces technologies pour améliorer le traitement des données non structurées et semi-structurées dans des applications d'IA générative.
