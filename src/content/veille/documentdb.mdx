# Résumé 
Microsoft a annoncé DocumentDB, une base de données NoSQL open-source conçue pour être compatible avec MongoDB et construite sur PostgreSQL. L’objectif est de créer une norme open-source pour les bases de données documentaires, favorisant l’interopérabilité et la flexibilité entre différentes solutions NoSQL. DocumentDB est sous licence MIT, garantissant une grande liberté d’utilisation et de modification.
